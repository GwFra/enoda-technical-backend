## Description

Enoda Technical Challenge

## Installation

```bash
$ npm install
```

## Setup

```bash
$ cp .env.defaults .env
```

Upon creation of the .env populate the keys with valid values

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
